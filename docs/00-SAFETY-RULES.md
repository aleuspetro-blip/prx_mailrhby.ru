# Правила безопасности проекта prx.mailrhby.ru

Дата начала проекта: 20.07.2026

## Критическое ограничение

На сервере работают действующие скрипты и сервисы. Любое вмешательство в их работу запрещено.

Запрещено без отдельного явного разрешения владельца:

- перезагружать или выключать сервер;
- перезапускать, останавливать, отключать или удалять службы, процессы, контейнеры и рабочие скрипты;
- изменять существующие конфигурации;
- обновлять или удалять пакеты и системное программное обеспечение;
- изменять firewall, маршрутизацию, DNS, веб-серверы, PHP, базы данных и задания cron;
- изменять права, владельцев или содержимое существующих файлов;
- выполнять команды, способные создать заметную нагрузку или повлиять на доступность сервера.

## Порядок работы

1. Работа с сервером выполняется только через консольные команды, передаваемые владельцу.
2. На этапе обследования используются только команды чтения и безопасной диагностики.
3. Владелец выполняет команду и передаёт её полный вывод для анализа.
4. Для просмотра и загрузки файлов используется FTP. Консольные редакторы не используются.
5. Перед любым будущим изменением сначала создаётся план, проверяется влияние на рабочие процессы и предусматривается откат.
6. Документация проекта и подтверждённые результаты сохраняются в этом репозитории.

## Текущий этап

Безопасная инвентаризация сервера и окружения домена `prx.mailrhby.ru` без внесения изменений.
