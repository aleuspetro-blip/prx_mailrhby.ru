# Результат установки AmneziaWG

Дата: 20.07.2026

## Статус

AmneziaWG успешно установлен на новом VPS Ubuntu 24.04.

Подтверждено:

- установлен пакет `amneziawg`;
- установлен DKMS-модуль `amneziawg/1.0.0` для ядра `6.8.0-63-generic`;
- команды `awg` и `awg-quick` доступны;
- модуль `amneziawg` загружается;
- временный интерфейс типа `amneziawg` успешно создаётся и удаляется;
- служба `awg-quick@wg0.service` остаётся выключенной и неактивной;
- основной маршрут сервера не изменён;
- публичный адрес VPS остаётся `185.237.186.95`.

## Следующий этап

1. На сайте Hide.mn создать конфигурацию для `AmneziaWG 2.0` с зарубежной локацией.
2. Скачать конфигурационный файл локально.
3. Передать его на VPS по SFTP во временный закрытый путь `/root/hidemy-awg.conf`.
4. Не запускать `awg-quick` до проверки конфигурации.
5. Проверить структуру файла с маскированием `PrivateKey`, `PublicKey` и `PresharedKey`.
6. Подготовить split-tunnel: через VPN должны идти только сети Telegram, а SSH и входящий MTProto-трафик должны оставаться на основном интерфейсе `eth0`.

## Ограничения безопасности

- Конфигурационный файл содержит секретные ключи и не должен попадать в GitHub, чат или публичные каталоги.
- Нельзя запускать полную конфигурацию с `AllowedIPs = 0.0.0.0/0` на удалённом VPS до адаптации маршрутизации.
- Не выполнять `systemctl enable awg-quick@wg0` до завершения тестов.
