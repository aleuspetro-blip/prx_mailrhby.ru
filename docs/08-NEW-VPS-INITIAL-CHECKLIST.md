# Первичная проверка нового VPS

Дата этапа: 20.07.2026

## Назначение сервера

Новый VPS с root-доступом предназначен для личного прокси Telegram. Предварительная архитектура: MTProto Proxy на TCP 443, позднее возможно добавление резервного транспорта на UDP 443.

## Правила этапа

На этапе первичной проверки выполняются только команды чтения и безопасные сетевые тесты. Запрещены установка и обновление пакетов, изменение firewall, перезапуск служб, изменение SSH, DNS и сетевых параметров.

## Что необходимо подтвердить

- фактическую ОС и версию ядра;
- тип виртуализации;
- доступные CPU, RAM, swap и диск;
- публичные IPv4/IPv6 и сетевые маршруты;
- занятые TCP/UDP-порты, особенно 443;
- активные службы и ошибки systemd;
- локальные правила UFW, nftables и iptables;
- наличие Docker/Podman и средств сборки;
- корректность времени и синхронизации;
- исходящий HTTPS-доступ к официальным ресурсам Telegram;
- текущее DNS-состояние prx.mailrhby.ru.

## Следующий этап

После анализа вывода составить план установки без конфликтов с существующими службами и определить, использовать Teleproxy, Telemt или другую проверенную реализацию.
