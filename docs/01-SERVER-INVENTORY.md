Основной маршрут:

- default gateway: `92.53.96.1`;
- интерфейс: `lan0.2`.

На узле назначено множество публичных IPv4-адресов разных сайтов и клиентов. Это ещё один сильный признак общего хостинга. Также присутствуют внутренние сети:

- `172.16.0.0/21`;
- `192.168.0.0/20`;
- маршрут к `192.168.0.0/16`.

На этом этапе нельзя считать какой-либо из показанных IP адресом проекта `prx.mailrhby.ru`. Домен подключён через CDN, поэтому отдельно потребуется определить DNS-цепочку, адрес origin и способ привязки домена внутри панели хостинга.
